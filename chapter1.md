# 行銷簡介

## 行銷的定義：

美國市場行銷學(American Marketing Association ， AMA) : 規劃並實施商品、理念或服務的定價、促銷、分銷，並創造交易來滿足個人或組織的目的的過程。

現代行銷學之父菲利普．科特勒(Philip Kotler) :「 是個人和集體透過創造，提供出售並與別人交換和價值，以獲得其所需所欲之物的一種社會和管理過程。」

管理學之彼得‧杜拉克(Peter Drucker) : 「行銷的目的，在於使銷售成為多餘，是要充分認識和了解顧客，使產品或服務能真正符合顧客的需求。」

行銷學者蓋瑞．阿姆斯壯(Gary Armstrong) : 「行銷是為顧客創造價值， 也是提升顧客生活品質的藝術。」

顧客關係管理大師克里斯琴．格隆羅斯(Christian Gronroos) :「行銷是在一種利益之上下，透過相互交換和承諾、建立、維持、鞏固與消費者與其他參與者的關係，實現各方的目地。」

王道行銷3.0作者王寶玲 ：「行銷簡單的說就是一個動人的故事，然後把事情搞大。」

行銷包含了銷售，但不等同銷售。

行銷的重點在於把產品變得好賣，而銷售則是創造最佳的銷售業績。銷售是業務主動大量開發客戶，尋找到準顧客；而行銷是透過大量的曝光，直接讓準顧客來找你，這就是行銷的魅力和威力。

行銷的目地 : 在於創造慾望，滿足需求。

1. 馬洛斯的需求理論 ：

  - 生理需求
  - 安全需求
  - 愛與歸屬感的需求
  - 尊重的需求
  - 自我實現的需求

2. ERG需求理論 :

  - 生存需求
  - 關係需求
  - 成長需求

## 行銷導向的演進

- 生產導向
- 產品導向 : 過度重視產品品質而輕忽消費者需求的觀念，就是美國哈佛大學教授所說的: 「行銷近視症」。
- 銷售導向 : 由內而外。
- 行銷導向 : 由外而內。
- 社會行銷 : 例如麥當勞和肯德基的包裝袋均採用再生紙製作。

## 顧客與市場分析

### 消費者市場的特性 :

1. 需求多樣，購買的變動率高。
2. 購買數量少，購買頻率高。
3. 購買者多，分佈範圍廣。
4. 一時興起，衝動性購買。
5. 產品選擇多，需求彈性大。

### 組織市場的購買特性：

1. 重視業務人員銷售。
2. 購買者少，但購買數量大。
3. 買賣雙方的關係更為密切。
4. 購買者的地理分佈集中。
5. 需求的價格彈性大。
6. 主要的需求來自新增廠辦設備。
7. 專業化的購買。
8. 進行直接採購。
9. 以租賃的方式。
10. 是一種衍生的需求。

### 影響消費者購買行為的因素 ：

1. 社會因素 : 包括參考群體、家庭、角色和地位。
2. 文化因素 : 包括文化、次文化和社會階層。
3. 個人因素 ： 包括年齡和職業。
4. 心理因素：包括信念與態度、動機、認知、學習。

### 影響組織購買行為的因素包含 :

環境、組織、個人、人際因素。

## 問題在產品

### 產品生命週期

行銷大師李維特於1965年提出產品生命週期分四個階段：

1. 導入期 :快速掠取策略、快速滲透策略
2. 成長期 : 好的行銷策略可以採取:改善產品品質、調整產品售價、擴展新市場。
3. 成熟期 優的行銷策略可採取 : 提高使用率、擴大使用人數、產品改良、調整行銷組合。
4. 衰退期: 強調轉的策略可採取 : 割捨策略、不動策略。

### 新產品主要四種類型：

1.模仿新產品 2.改良新產品 3.換代新產品 4.完全新產品

## 有通路行銷才會暢通

行銷通路的成員主要可分成三類 :

1.經銷商2.代理商3.促成者

### 通路

通路的階層是指產品從製造商的工廠到最終消費者手中，中間經過的通路商數目。

1. 零階通路：製造商==>消費者
2. 一階通路 : 製造商==>零售商==>消費者
3. 多階通路 : 製造商==>批發商==>>零售商==>消費者

通路的類型 :

### 店頭賣場式通路

1. 連鎖門市
2. 獨立店
3. 量販店
4. 專業大賣場
5. 綜合賣場
6. 超市
7. 百貨公司
8. 購物中心
9. 捷運商店街
10. 市集
11. 另類通路

  - 自動販賣機
  - 餐廳
  - 福利社
  - 攤販
  - 商展
  - 檳榔攤
  - ### 非店頭賣場式通路

12. 電視購物
13. 郵購型錄
14. 手機通路
15. 多層次傳銷:傳銷和直銷
16. 網路商店

  - 在網路拍賣平台販售
  - 租用知名網路商場的交易平台
  - 借用知名網路交易平台將自己的商品上架銷售
  - 架設獨立的網路商店

## 品牌決定價值

桂格創辦人John Stauart曾說企業要分產的話，我寧可要品牌、商標或商譽，其他的廠房、大樓、產品、我都可以送給你。 宏碁前董事長施振榮先生曾提出著名的「微笑曲線」：企業要創造高的價值，只有靠兩種方式 ：1.靠研發和設計2.靠品牌和通路。

品牌建立多元的正面聯想性，企業應考慮 : 特質、個性、利益、企業價值、使用者。 品牌的四種呈現方式 : 圖案、文字、文字與圖案的組合、象徵人物。

## 80/20法則與長尾理論

- 80/20法則 : 80%的價值來自於20%的因子，其餘20%的價值來自於80%的因子。
- 長尾理論 : 通路夠大，非主流的、需求量小的商品「總銷量」也能夠和主流的、需求量大的商品銷量抗衡。
